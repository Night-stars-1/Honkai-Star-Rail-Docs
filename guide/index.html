# Python的安装

  • 点击绿色部分open in new window安装Python3.11
  • 将页面拉至底部，下载Windows installer (64-bit)
  • 点击安装包进行安装
  • 勾选Add Python 3.x to PATH

# 启动星穹铁道小助手

注意

如果你是Win10用户，建议先下载这个Windows Terminalopen in new window

  • 下载最新版Auto_Star_Railopen in new window
  • 解压文件，并进入文件夹
  • 复制文件夹路径
  • 右键开始菜单，选择终端管理员，输入cd 刚刚复制的路径
  • 输入pip config set global.index-url https://mirrors.aliyun.com/pypi/simple设置国内pip源
  • 输入pip install -r requirements.txt安装依赖
  • 输入Python Honkai_Star_Rail.py运行星穹铁道小助手 ps: 输入Python Honkai_Star_Rail.py -h可以看见命令行快速启动信息

# 快速启动

  • 下载地址快速启动open in new window
  • env.bat为依赖安装脚本
  • start.exe为快速启动脚本

ps: 所有文件请放在项目根目录执行

# 适合小白的启动方式

  • 我们不推荐这种启动方式，如果你按照上面的教程实在无法启动再使用
  • 下载QQ群open in new window的星穹铁道小助手.exe然后双击运行，他在整合包文件夹中
  • ps: 第一次启动可能会报错，请重新启动
